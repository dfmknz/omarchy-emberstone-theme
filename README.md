### Install:

- SUPER+ALT+SPACE
- Install/Style/Theme
- copy/paste `https://github.com/dfmknz/omarchy-emberstone-theme`

### From the terminal:

```bash
omarchy-theme-install https://github.com/dfmknz/omarchy-emberstone-theme.git
```
